Javascript array at

2023-03-10

At times you want to fetch the last element in an array. I've typically done it by the getting the element at the length of the array minus one

const a = [123,23,43,22]
a[a.length-1]

Today I was made aware that array has a method called at where you can access via index similar to above. At also support negative numbers which would fetch from the end of the array. The code above could also be written as 

          const a = [123,23,43,22]
a.at(-1)
