Numbering with CSS Counters

2023-03-04

I had a list with a number of steps needed in a pipeline I wanted to visualize to the user the steps involved.

<div>
  <div>1: Do thing</div>
  <div>2: Next thing</div>
  <div>3: Last thing</div>
</div>

You can utilize css counters if you want to keep the numbering part away from your markup (so similar to an ordered list)

.steps {
  counter-reset: stepsCounter;
}

.steps div::before {
  counter-increment: stepsCounter;
  content: "ðŸ¦¶ " counter(stepsCounter) ": ";
}


<div class="steps">
  <div>Do thing</div>
  <div>Next thing</div>
  <div>Last thing</div>
</div>
