Spelsylt8 game jam

2023-03-25

The game jam somewhat loosely associated with the Swedish tech podcast Kodsnack just finished it's eight jam Spelsylt 8.

For each jam I try to focus on different things and for this one I set out to try and:

  1. Make sure I had a working game BEFORE doing graphics and/or music (I tend to go very much the opposite route otherwise)
  2. Not make the game mechanics part complicated or time consuming
  3. Build in a tutorial as part of the first level

All in all I think I manage to follow these three goals so happy with that! I also manged to try and create the music and Garageband and draw things in Procreate so all in all a lot of new stuff to try out in the two week period the jam lasted.
